 Japanese Vocabulary Quiz

A simple and beginner-friendly website designed to help Japanese language learners practice basic Japanese vocabulary through a vocabulary list and an interactive quiz.

📌 About the Project

This project contains 15 basic Japanese vocabulary words along with a simple quiz to help learners review and test their understanding.

I created this project as part of my journey to learn Web Development and Japanese language learning.

✨ Features

- 📚 15 Japanese vocabulary words
- 📝 Simple vocabulary quiz
- 🎯 Interactive quiz experience
- 🎨 Beginner-friendly and simple user interface

🛠️ Technologies Used

- HTML – Used to create the structure of the website
- CSS – Used for styling and designing the webpage
- JavaScript – Used to add interactive quiz functionality
- AI Assistance – Used as a learning and development aid while creating the project

🎓 Learning Purpose

This project was created as a beginner learning project. I used AI assistance to help build the initial version and am currently working on understanding the code, especially JavaScript, and improving the project as I learn more about web development.

Through this project, I explored concepts such as:

- HTML structure
- CSS styling
- Classes and IDs
- Basic webpage design
- JavaScript-based interactivity

🚀 Future Improvements

I plan to improve this project as I continue learning JavaScript by adding:

- More Japanese vocabulary
- Score tracking
- Correct and incorrect answer feedback
- Next and Previous question buttons
- Restart Quiz option
- Different difficulty levels
- More interactive learning features
